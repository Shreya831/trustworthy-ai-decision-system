# 🧠 Self-Explaining AI System

## 🚀 Overview
This project implements a **Self-Explaining AI System** that not only predicts loan approval decisions but also explains *why* the decision was made.

Unlike traditional ML models, this system enhances **transparency and trust** by combining:
- Machine Learning predictions
- Feature-level explanations (XAI)
- Human-readable reasoning
- Consistency validation

---

## ✨ Key Features

- 🔍 Loan Approval Prediction (Random Forest)
- 📊 Feature Importance-based Explainability
- 🧠 LLM-style Reasoning Generation
- ✅ Consistency Check between model and reasoning
- 🎯 Interactive Gradio Web Interface
- 📉 SHAP-based visualization (optional module)

---

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- SHAP (Explainable AI)
- Gradio (UI)

---

## ⚙️ How It Works

1. User selects a data instance from the test dataset  
2. Model predicts loan approval or rejection  
3. System extracts top influencing features  
4. Generates human-like reasoning  
5. Validates consistency between explanation and model  
6. Displays results via an interactive UI
